## 🔧 技术细节

### API 版本说明

| 功能类别 | API 版本 | 状态 |
|----------|----------|------|
| 表格操作 | v3 | ✅ 稳定 |
| 工作表操作 | v3 | ✅ 稳定 |
| 数据操作 | v2 | ✅ 稳定 |
| 权限管理 | v1 | ✅ 稳定 |
| 筛选/筛选视图 | v3 | ✅ 稳定 |

| 保护范围 | v2 | ⚠️ 需特定参数 |
| 数据校验 | v2 | ⚠️ 需特定参数 |
| 云盘操作 | v1 | ✅ 稳定 |

### 关键发现

1. **sheet_id 获取**：必须通过 `list_sheets` API 获取，不是工作表名称
2. **range 格式**：`<sheetId>!<范围>` 如 `9e553d!A1:B5`
3. **权限 API**：使用 v1 版本 (`/drive/v1/permissions`)
4. **数据写入**：`values_prepend` 在指定范围上方插入行
